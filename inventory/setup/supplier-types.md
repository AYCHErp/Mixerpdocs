| ------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| Supplier Type Id   | This will be automatically generated.                                                                                          |
| Supplier Type Code | Enter an alpha-numeric code for this supplier type.                                                                            |
| Supplier Type Name | Enter the supplier type name.                                                                                                  |
